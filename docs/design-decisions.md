# 設計判断

- PATは保存済みトークンの接続テストを提供する。書き込み権限がAPIのリポジトリ情報から確認できない場合は「未確認」とし、書き込み操作時に再判定する。入力されたPATはURL・ログ・公開ファイルへ出さない。
- note用本文に表・Markdown画像・raw HTMLが含まれる場合は、対象行と手動対応を表示してコピーを止める。変換前の原文Markdownは別モードでコピーできる。
- 画像アセットの棚卸しはGit Tree・記事本文・image-status.jsonを照合する。削除対象は未参照ファイルだけに限定し、選択と確認なしの自動削除は行わない。
- AI画像生成APIは静的PWAへ秘密鍵を置かず、当面はプロンプトのコピーと利用者による生成・確認・登録を採用する。
- 公開ダッシュボードの予定は、公開待ち記事の接頭辞順と端末内の開始日時・公開間隔から算出し、status.jsonへ予定日を書き戻さない。iPhoneのバックグラウンド通知はWeb Push送信サーバーを追加するまで予定表示に留める。
